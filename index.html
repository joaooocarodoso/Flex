O que é 'display-flex'?

É uma propriedade CSS que define um contêiner flexível, organizando elementos filhos automaticamente em uma linha ou coluna

O que é 'justify-content?'

É uma propriedade CSS do módulo Flexbox (e Grid) usada para alinhar itens ao longo do eixo principal (main axis) de um contêiner

O que é 'aling-items?'

Define como os itens (filhos) de um contêiner Flexbox ou Grid se alinham no eixo transversal (perpendicular ao eixo principal).
